Welcome Page


Working on a software project and not really sure how to identify your classes
and objects?

We know Identifying classes and objects for a program can be a bit tricky. We
built this web application to help programmers decide if their "potential class"
is worthy of being called a class.

Your Web Tool Guide!


This tool is designed to help you ensure potential classes are indeed classes.

1) Before you start you should have a list of your potential classes

2) In the top right-hand corner of this page you will find a "Start Button"
This will take you to a web page to enter in your candidates

3) Once you've entered in all of your candidates, at the top right-hand corner
you will find another "Next Button", this will take you to the questionnaire
page where you will be asked to answer questions about each candidate.
Once done, press the Next Button

4) Here you will find given the results for each
candidate. A short description will appear for each candidate that will not be considered a class
